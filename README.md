 🎥 Live Object Detection & Tracing

This project is a real-time object detection and tracking web application using Streamlit, YOLOv8, and WebRTC.  
It detects objects from your webcam and displays them live with tracking annotations.

 Features

- Real-time object detection
- YOLOv8 object tracking
- Webcam integration
- Live video streaming
- Browser-based interface

 Technologies Used

- Python
- Streamlit
- streamlit-webrtc
- Ultralytics YOLOv8
- OpenCV
- PyAV
